# Resources

This folder can hold public resources that support the portfolio:

| Resource | Status |
| --- | --- |
| CV | Add public PDF or link |
| Publication list | Add verified citations |
| Research statement | Add when ready |
| Teaching or speaking materials | Add public material only |
| Recommendation links | Add only with permission |

## Safety Notes

Do not include:

- transcripts;
- private supervisor feedback;
- unpublished confidential research;
- client documents;
- credentials;
- private employment files.
